# Скриншоты

Здесь лежит Android-набор из RuStore (1080×1920) и iOS-набор в `ios/` (1320×2868).
Лендинг (`index.html` во всех локалях — ru/en/ja) показывает их в переключаемой горизонтальной галерее.

|--------------------------|-------------------------------------|---------|
| `01-launch.png`          | первый запуск (тема, палитра, язык) | светлая |
| `02-initial-screen.png`  | история открытых слов               | светлая |
| `03-search.png`          | поиск с результатами                | светлая |
| `04-details.png`         | карточка слова                      | светлая |

Рекомендации:
- вертикальные скриншоты телефона; галерея тянет их по ширине;
- набор намеренно смешивает светлую и тёмную темы: сам сайт теперь только светлый (Ink), и скриншоты — единственное место, где видно тёмную тему приложения;
- можно ужать через `pngquant` или `oxipng`, чтобы страница грузилась легче (не обязательно).

Добавите/уберёте кадры — поправьте секцию `#screenshots` в `index.html` во всех локалях
(блоки `<figure class="shot">`).

Иконку приложения положите в `../assets/icon.png` (квадратная, ≥256×256), а картинку для
превью в соцсетях — в `../assets/og.png` (≈1200×630).
