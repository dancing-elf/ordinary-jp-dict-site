# Скриншоты

Положите сюда PNG с устройства/симулятора под этими именами — лендинг (`index.html`) ссылается на них:

| Файл           | Что показать                  |
|----------------|-------------------------------|
| `search.png`   | экран поиска с результатами   |
| `card.png`     | карточка слова (с фуриганой)  |
| `reader.png`   | разбор предложения (чтение)   |
| `kanji.png`    | разбор кандзи                 |

Рекомендации:
- вертикальные скриншоты телефона (примерно 9:19.5) — под такое соотношение свёрстана сетка;
- одинаковая тема (все светлые или все тёмные) смотрится опрятнее;
- можно ужать через `pngquant` или `oxipng`, чтобы страница грузилась легче (не обязательно).

Добавите/уберёте кадры — поправьте секцию `#screenshots` в `index.html` (блоки `<figure class="shot">`).

Иконку приложения положите в `../assets/icon.png` (квадратная, ≥256×256), а картинку для
превью в соцсетях — в `../assets/og.png` (≈1200×630).
