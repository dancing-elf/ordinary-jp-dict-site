サポート

ヘルプ・不具合報告・ご要望。

辞書をご利用いただきありがとうございます。気になる点や改善のご要望は、以下の窓口からどうぞ。

辞書の内容の誤り

訳語や読みの誤り、項目の誤字、抜けている語 — これらはオープン辞書のデータで、修正はアプリではなく 各プロジェクト側で行われます。出典元へ報告してください。修正は次回の更新でアプリに反映されます （更新はおおむね半年に一度）。

Warodai →

ロシア語の語釈 — 大和露辞典。

JMdict →

英語の語釈。

各項目の出典は、単語カードと出典ページで確認できます。

アプリの不具合・ご要望

クラッシュ、フリーズ、検索の不調、使いにくい点 — あるいはご要望。これらは GitHub のトラッカーへ （GitHub アカウントが必要です）。

アプリの不具合 →

クラッシュ、フリーズ、正しく動かない。

アイデア・ご要望 →

足りないもの、もっと便利にできる点。

すべての報告 →

よくある質問

インターネットは必要ですか？

いいえ。検索・文の解析・音声はオフラインで動きます。ネットワークは外部リンクと匿名の利用統計に のみ使われます。

訳はどこから来ていますか？

ロシア語は大和露辞典（Warodai）、英語は JMdict、ほか固有名詞・漢字・ふりがな・アクセントは オープン辞書から。一覧とライセンスはトップページとアプリの 「出典」画面にあります。

辞書はどのくらいの頻度で更新されますか？

定期的に更新します。おおむね半年に一度、出典元の最新データを取り込みます。項目の誤りを見つけたら、 出典元（Warodai または JMdict）へ報告してください。修正は更新で 反映されます。

プライバシーはどうなっていますか？

検索した語句・開いた単語・履歴は端末から出ません。詳細は プライバシーポリシーへ。

自分の辞書を追加できますか？

はい。アプリは od-dict/1 形式の辞書をインポートできます — 形式の説明をご覧ください。
